1. ¿Qué es un Algoritmo?

Un algoritmo es, simplemente, un conjunto de pasos ordenados y precisos que permiten resolver un problema o lograr un objetivo. No es nada mágico ni exclusivo de las computadoras. Los usamos todos los días sin darnos cuenta.

Piensa en un algoritmo como una "receta de cocina". Si sigues los pasos correctamente, obtendrás el platillo deseado.

Receta de un pastel

1. Mezclar harina y huevos
2. Hornear 30 min
3. Decorar

Algoritmo informático

1. Entrada de datos
2. Proceso matemático
3. Mostrar resultado

Características clave: Preciso (sin ambigüedades), Definido (siempre el mismo resultado con los mismos datos) y Finito (tiene que terminar después de algunos pasos).

3. ¿Cómo representamos un algoritmo?

Hay dos formas muy comunes para "dibujar" o escribir algoritmos antes de programar: Pseudocódigo y Diagramas de flujo.

Pseudocódigo

Es una mezcla entre lenguaje humano y de programación. Muy fácil de leer.

// Algoritmo para sumar dos números
INICIO
   LEER num1, num2
   resultado = num1 + num2
   MOSTRAR resultado
FIN

Diagrama de flujo

Representación gráfica con figuras geométricas.

INICIO
📥 Leer num1, num2
resultado = num1 + num2
📤 Mostrar resultado
FIN

4. Las 3 estructuras fundamentales

Todo algoritmo, por complejo que sea, se construye combinando estas tres estructuras.

1. Secuencia

Acciones que se ejecutan una detrás de otra, en orden.

Paso 1 Paso 2 Paso 3

Ejemplo: Para lavarse las manos: 1. Abrir llave, 2. Enjabonar, 3. Enjuagar, 4. Cerrar llave.

2. Condicionales (Toma de decisiones)

Se ejecuta un bloque u otro dependiendo de una condición.

¿Tiene hambre?
SI
Cocinar
NO
Seguir trabajando

3. Ciclos (Repetición)

Repite un bloque de acciones mientras se cumpla una condición.

Mientras no estén cortadas las 5 zanahorias: Cortar una zanahoria

5. Ejemplo práctico: Calculadora de propinas

Vamos a simular un algoritmo real. Imagina que quieres calcular el 15% de propina en un restaurante. El algoritmo sería:

// Algoritmo CALCULAR_PROPINA
INICIO
   // 1. Preguntar el total de la cuenta
   LEER total_cuenta
   // 2. Calcular propina = total_cuenta * 0.15
   propina = total_cuenta * 0.15
   // 3. Mostrar la propina
   MOSTRAR "Debes dejar $" + propina + " de propina"
FIN

¡Pruébalo tú mismo! Introduce un valor y el algoritmo (implementado en JavaScript) lo resolverá.

💡 ¡Aquí aparecerá tu propina!

*El algoritmo sigue exactamente los pasos: leer, calcular, mostrar.
